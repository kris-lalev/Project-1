The two functions mentioned below are private member functions because they are only used inside calculateformulacellsreference().

Their purpose is to calculate the value of a cell be it number, formula or string or reference to another cell.. The result and index(current position in formula) are passed by reference, because their new values are needed in the main function. They return true or false depending on the success of the calculation. this function calculates the part of the formula which is a number.
